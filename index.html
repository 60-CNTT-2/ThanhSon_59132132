What is Ajax?

As defined in Wikipedia

  • AJAX is a group of interrelated web deverlopment techniques used to create interactive web applications or rich Internet applications. With Ajax, web applications can retrive data from the sever asynchronously in the background without interfering with the display and behavior of the existing page
